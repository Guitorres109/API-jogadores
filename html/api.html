📌 Sobre o Projeto

Esta API foi desenvolvida para fornecer informações sobre jogadores de futebol de forma simples, rápida e segura. Os dados são armazenados localmente em arquivos JSON e servidos através de um servidor Node.js com Express.

👀 Exemplo Real - Explorador de Jogadores

https://nttordino.github.io/API-JOGADORES/

🛠️ Tecnologias Utilizadas

Node.js

Ambiente de execução JavaScript no servidor.

Express

Framework para criação de APIs e rotas.

CORS

Permite requisições entre diferentes origens.

dotenv

Gerenciamento de variáveis de ambiente.

JSON

Banco de dados simples para armazenar jogadores.

🔐 Segurança

A API utiliza autenticação via API Key.

?key=DSI3_SESI

Ou via header:

x-api-key: DSI3_SESI

🌐 Rotas da API

GET /verificar

Verifica se o servidor está online.

{ "status": "success", "message": "Servidor está online" }

GET /api/jogadores/aleatorio

Retorna um jogador aleatório.

/api/jogadores/aleatorio?key=SUA_API_KEY

GET /api/jogadores/:nome

Busca jogador pelo nome.

/api/jogadores/neymar?key=SUA_API_KEY

GET /img/:arquivo

Retorna imagens armazenadas localmente.

/img/neymar.jpg

⚙️ Funcionamento Interno

• Dados carregados de um arquivo JSON
• Seleção aleatória de jogadores e imagens
• Middleware de autenticação com API Key
• Servidor identifica automaticamente o IP local

📦 Exemplo de Resposta

{ "status": "success", "message": "Jogador: Neymar\nIdade: 31\nTime: Al-Hilal\nNacionalidade: Brasil", "foto": "http://IP:PORT/img/neymar.jpg" }

🚀 Como Usar

1. Inicie o servidor Node.js

2. Acesse as rotas via navegador ou ferramentas como Postman

3. Inclua sua API Key nas requisições
